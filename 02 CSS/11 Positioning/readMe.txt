Position Values:
 1. Static - Default, not effected by top,bottom,left,right Values
 2. Relative - tblr causes to move from its normal Position
 3. Absolute - Positioned Relative to parent element(container)
 4. Fixed - Positioned Relative to viewport
 5. Sticky - Positioned based on scroll position

 z-index: higher the value, more preference.
 We use z-index: 1; to be able to see the element and -1 will hide it.
