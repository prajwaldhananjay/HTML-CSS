3 ways: 
  1. Color Name
  2. RGB - (0-255, 0-255, 0-255)
        RGB(0,0,0) - Black
        RGB(255,255,255) - White

  3. Hex - 6 or 3 chracters
         - starts with #
         first 2 char - red, next 2 - Green, last 2 - blue
         common colors: 000(black), fff(white), ccc(grey)
